# QtFfplayer

#### 介绍
基于Ffmpeg的ffplay工具，移植到Qt的播放器。

#### 软件架构

请查看当前目录下的Tutorial.md文件。

#### 功能说明

1. 选择文件播放视频。
2. 停止播放视频。
3. 音量大小调节。
4. 暂停播放功能。
5. 静音功能。
6. 截屏功能。


#### 使用说明

1. The version of SDL is 2.0.14.
2. The version of ffmpeg is.. I forget it. What do you like.
3. The test Qt version of author is 5.9.8.
